Teacup Prophecies App Support

Logging in

If you are having trouble logging in, make sure that you log in to the original way that you signed up for the app. If you originally signed in with an email address and not Facebook then make sure that you log in with the email address that you originally signed in with, or make sure that your facebook email address is not the same as the email that you originally signed in with if you did not sign up using facebook.

Syntax Errors or incomplete quotes.

We have ported the quotes of the Teacup Prophecies book into a database as correctly as possible. If you come accross one that seems cut off, first try to scroll the quote to make sure that your device has not cut off the end due to display constraints. If you still notice an issue then there might be one that slipped through the cracks, and if that is the case, we want to correct it, so please do not hesitate to send us a note about it.

Sharing on Facebook

The Facebook SDK does not allow a pre population of text to a post. The text has to be written in the moment. Sharing these quotes on Facebook is still possible however by copying and then pasting the text into the Facebook post window.

More Support

We really hope that you enjoy the Teacup Prophecies App, and hope that the quotes help to add clarity to your life everyday. If you have any other comments or concerns about how this app functions, please email our tech team through the button below. Thanks.
